## 核心公式

### KDRoi 选词公式
```
KDRoi = (搜索量 × CPC) / 优化难度KD
```
KDRoi 越高越值得做。用 Semrush 导出关键词，Excel 计算排序。

### Adsense 收入估算
```
月收入(美元) ≈ 30 × UV基数 × ECPM × PV/UV比 × 广告位数
```
欧美 ECPM 3-10美元，国内 <1美元，相差10-30倍。
